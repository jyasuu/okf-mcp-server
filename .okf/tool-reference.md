# MCP Tool Reference

## Read Tools

| Tool | Description |
|------|-------------|
| `okf_list_bundles` | List all registered bundles |
| `okf_list_concepts` | List concepts, filtered by prefix, type, or tag |
| `okf_read_concept` | Read a concept by ID |
| `okf_read_index` | Read or synthesize index.md |
| `okf_search` | Full-text search across concepts |
| `okf_get_backlinks` | Find concepts linking to a given concept |
| `okf_get_graph` | Get the full link graph |
| `okf_validate_bundle` | Validate OKF conformance |

## Write Tools

| Tool | Description |
|------|-------------|
| `okf_write_concept` | Create or update a concept |
| `okf_delete_concept` | Delete a concept |
| `okf_write_index` | Write/update an index.md |
| `okf_append_log` | Append log entries |
| `okf_add_citation` | Add a citation to a concept |
